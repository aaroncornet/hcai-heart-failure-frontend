Information

Because this application handles very sensitive patient data, it is imperative that we, as the developers of this application, have your trust in handling this data. It should be stated that NONE of the information provided to us is stored in any way after the prediction has been made. It is destroyed on our servers as soon as you see the results on your web page.

For that very reason this project is completely open source and all the source code is publicly visible.
